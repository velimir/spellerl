spellerl
=====

Small library to produce grammatically correct English phrase for a given a term.

Supported types:

- [X] Positive integers with scale up to 10^64.

Build
-----

    $ rebar3 compile

Test
-----

    $ rebar3 eunit

Dialyze
-----

    $ rebar3 dialyzer

Run
---

    $ _build/default/bin/spellerl
